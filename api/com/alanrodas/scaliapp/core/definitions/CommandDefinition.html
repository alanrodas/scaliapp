CommandDefinition

abstract class CommandDefinition extends AnyRef

Represents a command definition for any kind of command weather named or not.

This has two known concrete subclasses, FiniteValuesCommand and MultipleValuesCommand.

This class provides defines the name of the command and common arguments such as flags and arguments of the form --arg.
