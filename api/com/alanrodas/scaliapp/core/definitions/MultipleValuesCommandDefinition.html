MultipleValuesCommandDefinition

case class MultipleValuesCommandDefinition(name: String, minAmountValues: Option[Int], maxAmountValues: Option[Int], passedArgDefinitions: Iterable[ArgumentDefinition], passedFlagDefinitions: Iterable[FlagDefinition], callback: (MultipleValuesCommand) ⇒ Unit) extends CommandDefinition with Product with Serializable

This class represents a command definition for commands that receive a finite amount of values.
