NamedValue

case class NamedValue(name: String, value: String, defined: Boolean) extends AbstractValue with Product with Serializable

Represents a named value passed to a command.

This is a concrete implementation of AbstractValue for named values, that means, the values that are passed to a command that accepts a concrete amount of arguments.

This class also provides accessors for the name of the value and a boolean stating if the value was passed as an argument or if it is the default value.

name

The name of the value passed to the command

value

The value passed to the command

defined

true if it was defined by the user, false otherwise
