Flag

case class Flag(name: String, altName: Option[String], value: Boolean, defined: Boolean) extends AbstractArgument[Boolean] with Product with Serializable

This is a special case of AbstractArgument where the value is a Boolean.

This class provides the value method as a way to access the argument value.

name

The name of the argument or flag, weather short or long

altName

An additional optional name for the argument or flag. short if name was long, and long if it was short

value

The value of the current argument

defined

true if it was defined by the user, false otherwise
