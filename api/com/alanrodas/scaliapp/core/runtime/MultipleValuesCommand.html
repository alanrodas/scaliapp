MultipleValuesCommand

case class MultipleValuesCommand(command: String, values: List[UnnamedValue], passedFlags: Iterable[Flag], passedArgs: Iterable[Argument]) extends Command with Product with Serializable

This is a concrete implementation for CalledCommand for UnnamedValuedCommand. It adds the values that the command was called with as a set of UnnamedValue.
